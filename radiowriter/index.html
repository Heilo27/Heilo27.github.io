Finally, program your Motorola radios without booting into Windows. Set frequencies, channels, tones, and settings—all from your Mac.

Mac Users Deserve Better

Motorola's official CPS software only runs on Windows. That means Mac users have been stuck with workarounds: Boot Camp, virtual machines, borrowing a friend's PC, or just not programming their radios at all.

RadioWriter changes that. Native Mac app. No Windows license. No VM overhead. No compromises.

Everything You Need

Channel Programming

Set up channels with frequencies, names, and power levels. Import and export codeplugs for easy sharing.

Frequency Management

Configure TX and RX frequencies, offsets, and splits. Support for simplex and repeater configurations.

Tone & Squelch

CTCSS, DCS, and digital squelch codes. Full tone table support for seamless repeater access.

Native Mac App

Built specifically for macOS. No emulation, no Wine, no virtual machines. Just download and run.

Codeplug Support

Read, write, and edit codeplug files. Compatible with common formats for easy migration.

Open Source

Full source code available on GitHub. Audit it, modify it, contribute to it. No black boxes.

Questions & Answers

Is this legal?

Yes. RadioWriter is original software that communicates with Motorola radios using documented protocols. It is not a copy or derivative of Motorola's CPS software.

Do I need a programming cable?

Yes, you'll need the appropriate programming cable for your radio model. RadioWriter works with standard Motorola-compatible USB programming cables.

Will this brick my radio?

RadioWriter includes safeguards to prevent writing invalid configurations. However, always keep a backup of your working codeplug before making changes.

Is there a Windows version?

No. Motorola already provides CPS for Windows. RadioWriter exists specifically to fill the gap for Mac users.

Can I contribute?

Yes! RadioWriter is open source. Check out the GitHub repository to report issues, request features, or submit pull requests.

Who made this?

RadioWriter is built by HeiloProjects, an independent developer studio focused on native Apple platform apps.
